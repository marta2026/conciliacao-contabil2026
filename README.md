# 📊 Conciliação Contábil — Nutricash & MaxiFrota

Sistema web de conciliação contábil, rodando via Streamlit.

## 🚀 Deploy no Streamlit Cloud

1. Suba os arquivos `app.py` e `requirements.txt` no GitHub
2. Acesse [share.streamlit.io](https://share.streamlit.io)
3. Conecte o repositório e selecione `app.py`
4. Clique em **Deploy**

## ▶️ Rodar localmente

```bash
pip install streamlit
streamlit run app.py
```
